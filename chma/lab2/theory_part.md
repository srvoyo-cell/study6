# ЛАБОРАТОРНАЯ РАБОТА №2

## ИССЛЕДОВАНИЕ ПОГРЕШНОСТИ ЧИСЛЕННОГО ИНТЕГРИРОВАНИЯ ФУНКЦИЙ

---

## Цель лабораторной работы

Закрепление теоретических знаний о методах численного интегрирования функций.
Приобретение навыков использования численных методов при интегрировании функций.

---

## Постановка задачи

Рассмотрим способы приближенного вычисления определенных интегралов

[
I = \int_a^b f(x),dx
\tag{2.1}
]

для заданной на отрезке ([a,b]) функции (f(x)), основанные на замене интеграла конечной суммой

[
I_n = \sum_{k=0}^{N} c_k f(x_k)
\tag{2.2}
]

где:
(c_k) — числовые коэффициенты,
(x_k) — точки отрезка ([a,b]), называемые узлами квадратурной формулы.

[
\int_a^b f(x),dx \approx \sum_{k=0}^{N} c_k f(x_k)
\tag{2.3}
]

Разность

[
\Psi = \int_a^b f(x),dx - \sum_{k=0}^{N} c_k f(x_k)
\tag{2.4}
]

называется погрешностью квадратурной формулы.

---

Введем на ([a,b]) равномерную сетку с шагом (h):

[
\omega_h = { x_i = a + i h, ; i=0,\dots,N }, \quad Nh = b-a
]

и представим интеграл (2.1) в виде суммы интегралов по частичным отрезкам:

[
\int_a^b f(x),dx \approx \sum_{i=1}^{N} \int_{x_{i-1}}^{x_i} f(x),dx
\tag{2.5}
]

---

# Формулы численного интегрирования

## 1. Формула прямоугольников

На частичном отрезке:

[
\int_{x_{i-1}}^{x_i} f(x),dx \approx f(x_{i-1/2}) h,
\quad x_{i-1/2} = x_i - 0.5h
\tag{2.6}
]

На всём отрезке:

[
\int_a^b f(x),dx \approx \sum_{i=1}^{N} f(x_{i-1/2})h
\tag{2.7}
]

Погрешность:

[
|\Psi| \le \frac{h^2(b-a)}{24} M_2,
\quad M_2 = \sup_{x \in [a,b]} |f''(x)|
\tag{2.8}
]

---

## 2. Формула трапеций

На частичном отрезке:

[
\int_{x_{i-1}}^{x_i} f(x),dx \approx \frac{f(x_{i-1}) + f(x_i)}{2} h
\tag{2.9}
]

На всём отрезке:

[
\int_a^b f(x),dx \approx \sum_{i=1}^{N} \frac{f(x_{i-1}) + f(x_i)}{2} h
\tag{2.10}
]

Погрешность:

[
|\Psi| \le \frac{h^2(b-a)}{12} M_2
\tag{2.11}
]

---

## 3. Формула Симпсона

На частичном отрезке:

[
\int_{x_{i-1}}^{x_i} f(x),dx \approx \frac{h}{6}
\left[ f(x_{i-1}) + 4f(x_{i-1/2}) + f(x_i) \right]
]

На всём отрезке:

[
\int_a^b f(x),dx =
\frac{h}{6}
\left[
f_0 + f_N + 2(f_1 + \dots + f_{N-1})

* 4(f_{1/2} + \dots + f_{N-1/2})
  \right]
  \tag{2.12}
  ]

Погрешность:

[
|\Psi| \le \frac{h^4(b-a)}{2880} M_4,
\quad M_4 = \sup_{x \in [a,b]} |f^{(4)}(x)|
\tag{2.13}
]

---

# Оценка погрешности методом Рунге

Пусть порядок метода — (m).

Если:

[
I - I_h \approx C h^m
]

то для шага (h/2):

[
|I - I_{h/2}| \approx
\frac{2|I_{h/2} - I_h|}{2^m - 2}
\tag{2.15}
]

Абсолютная погрешность:

[
|I - I_{h/2}| \le \varepsilon
\tag{2.17}
]

Относительная погрешность:

[
100 \cdot \frac{|I_{h/2} - I_h|}{|I_{h/2}|}
\le
\frac{2^m - 2}{2}\varepsilon
\tag{2.18}
]

# Задание на лабораторную работу

Для заданной функции (f(x)) на ([a,b]):

### 1.

Вычислить интеграл тремя методами:

* прямоугольников
* трапеций
* Симпсона

с относительной погрешностью:

[
\varepsilon = 10^{-2}%
]

Начальный шаг:

[
h = \frac{b-a}{10}
]

---

### 2.

Вычислить интеграл аналитически и с использованием квадратурных формул.
Построить графики зависимости:

[
\ln|\Psi| \quad \text{от} \quad \ln h
]

Сравнить с теоретическими оценками (2.8), (2.11), (2.13).

---

### 3.

Построить график функции

[
F(x) = \int_a^b f(x,t),dt
]

с относительной погрешностью (10^{-2}%), используя формулу Симпсона.
