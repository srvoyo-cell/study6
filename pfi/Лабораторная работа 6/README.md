# Лабораторная работа № 6

## Тема работы

Создание простого веб-приложения на `PHP` с подключением к базе данных `MySQL`.

В проекте реализован учебный вариант, близкий к методическим указаниям:
- есть форма для добавления записи;
- есть поиск по критерию;
- есть вывод данных в таблицу;
- приложение можно запустить локально и открыть по локальной сети.

## Назначение приложения

Приложение хранит список спортсменов и позволяет работать с этими данными через браузер.

Пользователь может:
- добавить нового спортсмена;
- указать вид спорта;
- выполнить поиск по фамилии;
- выполнить поиск по виду спорта;
- просмотреть все записи из базы данных.

## Используемые технологии

- `PHP`
- `MySQL`
- `HTML`
- `CSS`
- встроенный веб-сервер `PHP`

## Файлы проекта

- [connection.php](/Users/michael/Documents/study6/pfi/Лабораторная работа 6/connection.php) — параметры подключения к базе данных.

## Требования для запуска

Для выполнения лабораторной работы нужны:
- установленный `PHP`;
- установленный `MySQL`;
- запущенный сервис `MySQL`.

На `macOS` установка через `Homebrew`:

```bash
brew install php mysql
brew services start mysql
```

Проверка:

```bash
php -v
mysql --version
```

## Подготовка базы данных

Перед первым запуском нужно один раз выполнить SQL-скрипт:

```bash
cd '/Users/michael/Documents/study6/pfi/Лабораторная работа 6'
mysql -u root < init.sql
```

Если для пользователя `root` установлен пароль:

```bash
mysql -u root -p < init.sql
```

После выполнения скрипта:
- создаётся база данных `lab6_webapp`;
- создаётся таблица `sportsmen`;
- добавляются 4 начальные записи.

## Настройка подключения к базе данных

Параметры подключения находятся в файле [connection.php](/Users/michael/Documents/study6/pfi/Лабораторная работа 6/connection.php).

По умолчанию используются:
- `host = localhost`
- `user = root`
- `password = ''`
- `database = lab6_webapp`

Если на вашем компьютере используется другой пользователь, пароль или имя базы, эти значения нужно изменить.

## Запуск приложения

Для запуска используется встроенный сервер `PHP`:

```bash
cd '/Users/michael/Documents/study6/pfi/Лабораторная работа 6'
php -S 0.0.0.0:8000
```

После запуска приложение будет доступно:
- локально: `http://127.0.0.1:8000`
- по локальной сети: `http://IP_вашего_компьютера:8000`

## Порядок работы с приложением

### Добавление записи

1. Открыть страницу в браузере.
2. Ввести фамилию спортсмена.
3. Ввести вид спорта.
4. Нажать кнопку `Добавить`.
5. Убедиться, что появилось сообщение об успешном добавлении и запись появилась в таблице.

### Поиск записи

1. Ввести в поле поиска фамилию или вид спорта.
2. Нажать кнопку `Найти`.
3. Проверить, что на странице отображаются только подходящие строки.

Если поле поиска оставить пустым, выводятся все записи.

## Структура базы данных

Используется таблица `sportsmen`.

Поля таблицы:
- `id` — идентификатор записи;
- `surname` — фамилия спортсмена;
- `sport` — вид спорта;
- `created_at` — дата и время добавления.

## Как работает код

В файле [index.php](/Users/michael/Documents/study6/pfi/Лабораторная работа 6/index.php):
- создаётся подключение к базе данных;
- принимаются данные форм методом `POST`;
- запись добавляется через `INSERT`;
- поиск выполняется через `SELECT ... LIKE`;
- результат выводится в HTML-таблицу.

Для SQL-запросов используются подготовленные выражения `mysqli`, чтобы избежать ошибок при передаче данных.

## Возможные ошибки

### Не удаётся подключиться к базе данных

Проверьте:
- запущен ли `MySQL`;
- выполнен ли файл `init.sql`;
- совпадают ли параметры в [connection.php](/Users/michael/Documents/study6/pfi/Лабораторная работа 6/connection.php).

Если `MySQL` не запущен:

```bash
brew services start mysql
```

### Занят порт 8000

Если порт `8000` занят, можно запустить приложение на другом порту:

```bash
php -S 0.0.0.0:8080
```
