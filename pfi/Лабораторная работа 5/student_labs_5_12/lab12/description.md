# Лабораторная работа 12

Тема: массивы.

Что сделано:
- Простая анимация движения круга по точкам из массива.
- Подсчёт количества положительных элементов массива.
- Поиск максимального элемента массива.

Как проверить:
1. Открыть `index.html`.
2. Нажать `Старт` для анимации.
3. Ввести массив чисел и нажать `Обработать массив`.
