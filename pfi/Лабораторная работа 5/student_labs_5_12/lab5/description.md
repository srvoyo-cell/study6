# Лабораторная работа 5

Тема: переключатели (radio buttons).

Что сделано:
- Создана простая анкета человека.
- Для выбора пола, возраста и семейного положения используются переключатели.
- После нажатия кнопки выводится краткая сводка по выбранным пунктам.

Как проверить:
1. Открыть `index.html` в браузере.
2. Выбрать значения в каждом блоке.
3. Нажать кнопку `Показать результат`.
