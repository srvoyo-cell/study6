# Лабораторная работа 10

Тема: объект `Date`.

Что сделано:
- По выбранной дате определяется номер недели в году.
- По дате рождения определяется знак зодиака.
- По году определяется название года по старояпонскому календарю.

Как проверить:
1. Открыть `index.html`.
2. Выбрать дату.
3. Нажать `Определить`.
