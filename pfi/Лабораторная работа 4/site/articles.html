Коротко про CSS

Здесь собрана простая информация по теме CSS. Эта страница нужна не только для текста, но и для показа разных заголовков.

На главную К графике

Основы CSS

CSS нужен для оформления сайта. С его помощью можно менять цвет, размер шрифта, отступы, фон и расположение элементов.

Селекторы

Селектор показывает, к какому элементу применяется стиль. Например, можно задать стиль для тега, класса или ссылки.

Псевдоклассы

Псевдоклассы используются для состояний элемента. Например, при наведении мыши ссылка может менять цвет.

Оформление страницы

Чтобы страница выглядела аккуратно, обычно задают общий фон, ширину блока, отступы и рамки. В этой работе используется самый простой вариант оформления.

  • CSS подключается в теге link.
  • Все страницы используют один и тот же файл стилей.
  • За счет этого оформление везде одинаковое.
