# Описание лабораторной работы №4

## Что сделано

Сделан простой учебный сайт из нескольких страниц:

- `index.html` - главная страница.
- `articles.html` - страница с короткой теорией по CSS.
- `gallery.html` - страница с графическими элементами.
- `contacts.html` - страница с итоговой таблицей и контактами.

Также сделаны:

- `css/styles.css` - общий файл стилей.
- папка `assets` с графическими файлами `svg`.

## Как сделано

1. Сначала была создана структура сайта из нескольких HTML-страниц.
2. Потом был подключен один общий файл стилей через тег `<link>`.
3. В CSS были оформлены:
   - заголовки `h1`, `h2`, `h3`, `h4`;
   - ссылки;
   - основной текст;
   - таблицы и ячейки;
   - меню с 4 уровнями вложенности.
4. Для графики были добавлены простые SVG-файлы:
   - баннер;
   - разделитель;
   - буллет;
   - иконки.
5. Сайт специально сделан простым, без сложного дизайна, чтобы он выглядел как обычная учебная работа.

## Что можно показать преподавателю

- Внешний CSS-файл: `css/styles.css`.
- Многоуровневое меню слева.
- Заголовки разных уровней на страницах.
- Таблицы на главной странице и на странице контактов.
- Графические элементы в папке `assets`.

## Возможные вопросы и ответы

### 1. Почему стили вынесены в отдельный файл?

Потому что так удобнее. Один файл стилей можно подключить ко всем страницам, и тогда оформление будет одинаковым на всем сайте.

### 2. Как подключается CSS к странице?

CSS подключается в разделе `<head>` с помощью тега:

```html
<link rel="stylesheet" href="css/styles.css">
```

### 3. Зачем нужно меню 4 уровней?

Это одно из требований лабораторной работы. Оно показывает, что я умею делать вложенные списки и оформлять разные уровни меню по-разному.

### 4. Как сделано меню?

Меню сделано через вложенные списки `<ul>` и `<li>`. Для каждого уровня используются свои классы, например `menu-level-1`, `menu-level-2` и так далее.

### 5. Где на сайте есть таблицы?

Таблица есть на главной странице и на странице `contacts.html`. Они нужны для показа оформления таблиц и ячеек.

### 6. Зачем нужны SVG-файлы?

SVG удобны тем, что это легкие векторные изображения. Они не теряют качество при увеличении и хорошо подходят для простых иконок и баннера.

### 7. Что такое псевдокласс?

Псевдокласс - это специальное состояние элемента. Например, `:hover` срабатывает, когда курсор мыши находится над ссылкой.

### 8. Какие псевдоклассы здесь используются?

В этой работе используются `a:hover` и `a:visited` для ссылок.

### 9. Почему сайт сделан простым?

Сайт упрощен специально, чтобы он выглядел как учебная работа студента, а не как профессиональный коммерческий сайт.

### 10. Что можно изменить перед сдачей?

Можно поменять текст, e-mail, названия разделов и тему сайта. Основная структура и оформление уже готовы.
