Для чего сделан этот сайт

Сайт нужен для демонстрации того, что я умею подключать внешний CSS и оформлять основные элементы страницы.

Что можно показать преподавателю

  • Файл styles.css с общими стилями.
  • Многоуровневое меню слева.
  • Отдельные файлы графики в папке assets.

Сайт сделан специально простым, без сложного дизайна, чтобы он выглядел как обычная студенческая работа.
